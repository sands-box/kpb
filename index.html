Pengertian Turunan Parsial

Turunan parsial adalah turunan dari fungsi dengan beberapa variabel terhadap salah satu variabelnya, dengan menganggap variabel lainnya sebagai konstanta.

Jika kita memiliki fungsi f(x, y), maka:

fx(x, y) = ∂f/∂x = turunan f terhadap x (y dianggap konstan)
fy(x, y) = ∂f/∂y = turunan f terhadap y (x dianggap konstan)

Notasi Turunan Parsial

Ada beberapa notasi untuk turunan parsial:

∂f/∂x, fx, Dxf
∂f/∂y, fy, Dyf

Rumus Turunan Parsial

Aturan turunan parsial sama dengan aturan turunan biasa, hanya saja kita memperlakukan variabel lain sebagai konstanta.

Contoh Perhitungan

Misalkan f(x, y) = 3x²y + 2xy³

∂f/∂x = 6xy + 2y³ (y dianggap konstan)
∂f/∂y = 3x² + 6xy² (x dianggap konstan)

Aturan Rantai untuk Turunan Parsial

Jika z = f(x, y) dengan x = g(t) dan y = h(t), maka:

dz/dt = (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt)

Aplikasi Turunan Parsial

Turunan parsial memiliki banyak aplikasi dalam berbagai bidang:

1. Optimasi Multivariabel

Digunakan untuk mencari nilai maksimum atau minimum dari fungsi dengan beberapa variabel.

2. Ekonomi

Dalam ekonomi, turunan parsial digunakan untuk menghitung produk marginal, elastisitas, dll.

3. Fisika

Dalam termodinamika, turunan parsial digunakan untuk menghitung perubahan keadaan sistem.

4. Machine Learning

Digunakan dalam gradient descent untuk optimisasi model machine learning.

Contoh 1: Turunan Parsial Dasar

Tentukan turunan parsial pertama dari fungsi f(x, y) = x³ + 2x²y - y²

Penyelesaian:

Untuk ∂f/∂x, kita anggap y sebagai konstanta:

∂f/∂x = 3x² + 4xy

Untuk ∂f/∂y, kita anggap x sebagai konstanta:

∂f/∂y = 2x² - 2y

Contoh 2: Fungsi Trigonometri

Tentukan turunan parsial pertama dari fungsi f(x, y) = sin(x²y) + exy

Penyelesaian:

Untuk ∂f/∂x:

∂f/∂x = cos(x²y) · 2xy + exy · y
= 2xy cos(x²y) + y exy

Untuk ∂f/∂y:

∂f/∂y = cos(x²y) · x² + exy · x
= x² cos(x²y) + x exy

Contoh 3: Aturan Rantai

Jika z = x² + y², dengan x = 2t dan y = 3t, tentukan dz/dt menggunakan aturan rantai.

Penyelesaian:

Pertama, hitung turunan parsial:

∂z/∂x = 2x
∂z/∂y = 2y

Kemudian hitung dx/dt dan dy/dt:

dx/dt = 2
dy/dt = 3

Gunakan aturan rantai:

dz/dt = (∂z/∂x)(dx/dt) + (∂z/∂y)(dy/dt)
= (2x)(2) + (2y)(3)
= 4x + 6y

Substitusi x = 2t dan y = 3t:

dz/dt = 4(2t) + 6(3t) = 8t + 18t = 26t
